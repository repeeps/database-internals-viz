# 데이터베이스, 눈으로 보기 (Database Internals, Visualized)

수식도 코드도 없이, **데이터베이스 엔진 내부**를 눈으로 따라가는 인터랙티브 학습 사이트.
모든 동작(B+tree 탐색·쿼리 비용·트랜잭션 타임라인·버전 사슬)은 브라우저에서 실시간으로 계산·렌더되고, 특정 제품(PostgreSQL·MySQL 등)에 매이지 않은 **일반 원리**로 설명합니다.

👉 **[index.html](index.html)** 을 브라우저로 열면 바로 시작 (빌드 불필요).

## 무엇을 다루나

두 축이 전부예요 — **빨리(그리고 똑똑하게) 찾기**와 **안전하게 바꾸기**.

| PART | 챕터 | 내용 |
|------|------|------|

| **A · 저장과 읽기** | 01–02 | 디스크는 느리다(페이지·버퍼풀) · 비용의 언어·풀 스캔 |
| **B · 인덱스** ★ | 03–06 | 정렬의 힘 · B+tree · 범위·정렬 · 인덱스의 비용·모양 |

## 결

- **수식 없는 직관 입문** — 4스텝(직관 → 인터랙션 → 심화 → 정리)으로 한 챕터씩.
- **SQL 강좌가 아니라 멘탈 모델** — 보이지 않는 엔진 속 동작을 그림으로.
- **자매 트랙 연결** — 웹의 FastAPI(쿼리·N+1·커넥션풀), 동시성(잠금·경쟁·데드락), 알고리즘(이진탐색·B-tree·해시)의 바로 그 아래 계층.
- **다크 neural-lab 테마** · 모든 그림은 SVG로 실시간 렌더.
